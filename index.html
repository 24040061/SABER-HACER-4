¿Qué es JavaScript?

Usando Selectores, Eventos y Funciones

JavaScript

JavaScript es el lenguaje que permite agregar interactividad a las páginas web. Con él podemos responder a acciones del usuario

, cambiar el contenido, o crear animaciones. Los eventos de JavaScript son señales que indican que algo ha ocurrido en una página web

, como interacciones del usuario (clics, movimientos del ratón) o eventos del sistema (carga de la página)

1. Selectores

Los selectores permiten acceder a elementos del HTML usando su etiqueta, clase o id.

const titulo = document.getElementById("titulo");
const botones = document.querySelectorAll(".btn");

2. Eventos

Los eventos permiten ejecutar una acción cuando el usuario hace algo, como un clic o escribir texto.

boton.addEventListener("click", saludar);

3. Funciones

Las funciones son bloques de código que realizan una tarea. Podemos llamarlas varias veces.

function saludar() {
  alert("¡Hola, programador!");
}
